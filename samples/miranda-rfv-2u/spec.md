# Miranda RFV — 2U Liquid-Cooled Network Server
## Техническое задание v1.1 (фиксация от 02.05.2026)

Назначение: учебно-демонстрационный стенд для роботизированной системы контроля качества (RFID + умные камеры). Макет полностью разборный, с намеренно сохранённой сложностью сборки (2–3 сборочные станции, прецизионное позиционирование).

## 1. Общие требования

| Параметр        | Значение                          |
|-----------------|-----------------------------------|
| Форм-фактор     | 2U rack-mount chassis             |
| Высота          | 88,9 мм                          |
| Ширина (с ушами)| 482,6 мм                         |
| Глубина         | **780 мм** (фиксация, ИТ.1 выбрал 780 в вилке 770–790)|
| Сборка          | полностью разборный, 2–3 станции|
| Tolerance       | ±0,05–0,1 мм на ключевых сопряжениях |
| RFID/QR площадки | на всех ключевых деталях |
| Alignment pins  | dowel pins с прецизионной посадкой |

## 2. Состав макета

### 2.1. Шасси (Chassis)

- Материал: оцинкованная сталь, толщина листа **1,0 мм** (фиксация).
- Открытая верхняя крышка для демонстрации.
- Монтажные уши 19".
- Направляющие для плат.
- Отверстия под винты M3/M4.
- Задняя панель с вырезами под bus bar и QDC.
- Внутренние крепёжные бобышки и alignment pins.

### 2.2. Основная модульная ASIC-плата

- Модульная конструкция: центральная interconnect-плата + 2–4 compute-платы.
- По умолчанию: **2 ASIC, footprint 80×80 мм** (фиксация).
- Легко поменять на 4 через UserParameter `asic_count`.
- Board-to-board коннекторы + минимум **4 alignment pins на каждое соединение**.

### 2.3. Плата управления (BMC)

- Отдельная плата справа от основной.
- BMC-контроллер (placeholder), зелёные модули, разъёмы, индикаторы.
- Независимое крепление с alignment pins.

### 2.4. Медные cold plates («башни» охлаждения)

- По одной на каждый ASIC, split-type.
- Материал: медь **C11000** (ETP).
- Микроканалы внутри + U-/Г-образные медные трубки.
- Flatness площадки контакта ≤ 0,05 мм.

### 2.5. Центральный manifold

- Ровно по центру.
- Blind-mate QDC фитинги.
- Медь C11000 или нержавеющая сталь (решается в модуле manifold).

### 2.6. Шланги / трубки

- Отдельная группа деталей (гибкие + жёсткие).
- Цвета: синие (подача), чёрные (обратка).
- Quick-disconnect коннекторы.

### 2.7. Bus bar (54 V HVDC)

- Толстый медный bus bar сзади по центру.
- Номинал: **54 V HVDC** (фиксация).
- Сечение подбирается по пиковому току (расчёт в модуле bus_bar).

### 2.8. Фронт-панель

- Съёмная передняя панель 2U.
- Порты: 3×Ethernet, индикаторы, USB управления.
- Крепление: винты + направляющие.
- RFID-метки и контрольные точки для камер.

## 3. Дерево сборки

```
Miranda_RFV_2U (top assembly)
├── Chassis
│   ├── Base
│   ├── Side_L / Side_R
│   ├── Rear
│   └── Rack_Ear_L / Rack_Ear_R
├── ASIC_Board (interconnect + 2×compute)
├── BMC_Board
├── Cold_Plate × 2 (split-type)
├── Manifold
├── Hose_Set (синие + чёрные)
├── Bus_Bar_54V
└── Front_Panel
```

## 4. Координатная система

- Начало координат — центр передней кромки базовой пластины.
- X — вправо (по ширине), Y — вглубь (depth), Z — вверх.
- Все компоненты позиционируются от этой точки через alignment pins.

## 5. Экспортные форматы

На каждый deliverable производится:

- `.f3d` — native Fusion 360 archive с timeline и UserParameters (source-of-truth);
- `.step` — B-Rep для интероперабельности (SolidWorks, AutoCAD);
- `.stl` — для 3D-печати демо-макета.

## 6. Открытые вопросы (ожидают решения в следующих спринтах)

1. Пиковый ток bus bar (для расчета сечения меди).
2. Материал manifold (медь vs нерж.).
3. Геометрия микроканалов cold plate (skived fin / 3D-печать?).
4. Конкретная модель BMC (влияет на разводку).
5. RFID-метки: HF (13,56 МГц) или UHF (860–960 МГц)? Размер площадок зависит от выбора.
